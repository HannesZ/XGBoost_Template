# XGBoost_Template
An XGBoost Template to build powerful actuarial models.

Given raw tabular data in the form of a data.frame, this template will give the user a ready to go manual for creating 
xgboost-models which proof to be just as could or better than ordinary actuarial models like GLM. Once created, the user can then apply the model with the subsequent code which contains unit-tests to avoid dreaded mlr-errors.
